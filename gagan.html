HTML Theory: Image & Paragraph Tags

1. The `` Tag

The <img> tag is used to embed an image in an HTML page. It is a self-closing tag (meaning it does not need a separate closing tag like </img>).

  • src attribute: Specifies the URL or file path of the image source. In the example above, a direct link to an image on Wikipedia is used.
  • alt attribute: Provides "alternate text" for the image. This text is displayed if the image cannot be loaded and is crucial for accessibility (screen readers use it to describe the image content).
